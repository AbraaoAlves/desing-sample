# desing-sample


##Dev Tools:

1. [install nodejs](http://nodejs.org) 
2. [install browser-sync](http://www.browsersync.io/):
```	
npm install -g browser-sync
```	

## Run site:
```	
cd src
browser-sync start --server --files "*.html, *.css, *.js"
```	

## Deploy site in github:

```	
npm run deploy
```
